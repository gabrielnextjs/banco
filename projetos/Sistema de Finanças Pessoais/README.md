# Sistema de Financas Pessoais

## Visao geral
Projeto em desenvolvimento para controle financeiro pessoal no terminal.
A ideia e registrar receitas e despesas e acompanhar saldo e resumo mensal.

## Escopo planejado
- Adicionar receita.
- Adicionar despesa.
- Listar lancamentos.
- Exibir resumo do mes (receitas, despesas e saldo).
- Salvar dados em arquivo ao sair.

## Regras planejadas
- Valor deve ser numerico e maior que zero.
- Cada lancamento deve ter descricao e valor.
- O saldo e atualizado a cada novo registro.
