# Sistema de Cadastro de Membros

## Visao geral
Aplicacao de terminal para cadastro e listagem de membros.
Projeto focado em listas, dicionarios e menu interativo.

## O que o sistema faz
- Menu para adicionar membro, listar membros e sair.
- Cada membro e salvo com nome e nivel.
- Mostra mensagem quando nao ha membros cadastrados.

## Regras principais
- Nivel do membro e informado como numero.
- Dados ficam em memoria durante a execucao.
- Encerrando o programa, a lista atual e perdida.

## Como executar
`ash
python main.py
`
